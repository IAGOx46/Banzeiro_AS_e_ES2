# Interfaces Sistema Banzeiro
Nesta seção são apresentadas as telas do sistema Banzeiro em execução, demonstrando o funcionamento real do sistema após a implementação das interfaces.
As capturas ilustram o fluxo principal do usuário, desde as Boas vindas, o login e cadastro, até o dashboard, consulta de dados climáticos,informações sobre o nível dos rios e histórico.

As telas evidenciam como o protótipo foi transformado em um sistema funcional, mantendo a identidade visual planejada no Figma e integrando recursos essenciais para o monitoramento climático.
